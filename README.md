# resume-templates

A growing collection of **resume and CV templates** in different structures and styles. Pick a folder, customize the content, and export to PDF or host as a static page.

## Templates

| Folder | Format | Notes |
|--------|--------|--------|
| [`single-page-html`](./single-page-html/) | HTML (single file) | Gradient header, skills grid, experience, print-friendly A4. No build step. |

More layouts (e.g. two-column HTML, minimal, alternate typography) can be added as **new top-level folders** next to `single-page-html`.

## Using this repository

1. Clone or download the repo.
2. Open the README inside the template folder you want (e.g. `single-page-html/README.md`).
3. Follow that template’s instructions for placeholders and PDF export.

## GitHub Pages

If you publish the whole repo: set the site **root** to the repository, then open  
`https://YOUR_USERNAME.github.io/resume-templates/single-page-html/`  
(or configure **Pages** to publish from the `single-page-html` folder if your host supports a subfolder as the doc root).

## Contributing

Add a new template by creating a **new directory** (e.g. `two-column-html/`) with its own `README.md`, `index.html` (or entry files), and any assets—keep each format self-contained.
